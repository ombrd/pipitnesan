# Pipitnesan Project

This repository contains the full source code for the Pipitnesan project, which includes both the backend/web application and the mobile application.

## Project Structure

- **[`backend/`](./backend)**: Contains the Laravel-based backend APIs and web management portal.
- **[`mobile_app/`](./mobile_app)**: Contains the React Native mobile application for end users.

## Getting Started

To get started with development, you can follow the specific instructions in each module's respective `README.md` file.

- For backend setup and API documentation, refer to the [Backend README](./backend/README.md).
- For mobile app setup and run instructions, refer to the [Mobile App README](./mobile_app/README.md).

## Prerequisites

Before running the projects, make sure you have the following installed on your machine:
- PHP & Composer (for the backend)
- Node.js & npm/yarn (for the mobile app)
- Android Studio / Xcode (for mobile app compilation and simulation)
- Docker (optional, for containerized backend environment in the future)

## Version Control

This repository is unified at the parent directory. Both backend and mobile app share the same git history. Make sure to commit at this root level directory.
